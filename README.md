🚀 Job Analyzer Platform

**The Job Analyzer Platform** is a premium, full-stack application designed to solve the "black hole" of job hunting. It empowers professionals to analyze job market data, track application lifecycles, and secure their professional identity with an elite, AI-driven interface.

💎 Features That Set This Project Apart

🧠 AI-Driven Insights
Smart Analysis**: Utilizes AI logic to evaluate role compatibility and "force" high-value opportunities to the top.
**Premium Dashboard**: A high-fidelity, violet-themed command center featuring real-time data visualization.

### 🔒 Enterprise-Grade Security

* **Supabase Integration**: Robust authentication with protected routes and identity-secured data handling.

### 📱 Modern UX/UI

* **PWA Ready**: Fully configured with `site.webmanifest` and custom brand assets for mobile installation.
* **Responsive Design**: Mobile-first architecture built with Tailwind CSS and advanced CSS animations (Pulse/Glow effects).

## 🚀 Quick Start for Reviewers

To run this project locally and explore the code:

### 1. Clone the repository

```bash
git clone https://github.com/your-username/job-analyzer-platform.git
cd job-analyzer-platform/client

```

### 2. Configure Environment

Create a `.env` file in the `/client` directory:

```env
PORT=3000
REACT_APP_SUPABASE_URL=your_supabase_url
REACT_APP_SUPABASE_ANON_KEY=your_supabase_key

```

### 3. Install & Launch

```bash
npm install
npm start

```

*App will be accessible at `http://localhost:3000`
